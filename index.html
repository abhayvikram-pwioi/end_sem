JavaScript Web Developer (Advanced JS) - Question Paper Set 2

Total Marks: 60 | Time: 3 Hours

Part A (30 Marks)

Answer any 6 questions. Each question carries 5 marks.

Q1.Create an HTML page with a temperature converter (Celsius to Fahrenheit and vice versa). Use input field, and convert button. Implement using function expressions and demonstrate. Style inputs and results nicely.

[5 Marks]

Q2. Build a color palette generator. Have a button that generates 5 random colors (using Math functions). Display them as colored divs with hex codes. Allow clicking a color to copy to clipboard.

[5 Marks]

Q3. Implement a counter box with increment, decrement, and reset buttons. Add a feature to set a custom step value. Persist count using localStorage. Use let/const properly and show understanding of block scope inside event handlers.

[5 Marks]

Q4. Create a searchable dropdown using a datalist. Populate it with an array of programming topics from the course (e.g., DOM, Functions, Arrays). On selection, display detailed explanation fetched from a Map.

[5 Marks]

Q5. Design a simple form for student registration. Use try-catch for validating email format and age (>18). On successful submission, add to an array and display in a table. Clear form after submit. Use CSS for validation error messages.

[5 Marks]

Q6. Build tabs interface (Home, About, Contact) using HTML divs and JS. Switching tabs should load different content (use objects for tab data).

[5 Marks]

Part B (30 Marks)

Answer all 2 questions. Each carries 10 marks.

[10 Marks]

Q8. Build a "Shopping Cart Simulator". Display products from an array. Add to cart functionality with quantity. Show cart sidebar with total calculation. Use OOP principles (constructor for Product and Cart). Persist cart. Implement discount code validation (try-catch + conditionals).
