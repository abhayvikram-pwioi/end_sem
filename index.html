JavaScript Web Developer (Advanced JS) - Question Paper Set 2

Total Marks: 60 | Time: 3 Hours

Part A (30 Marks)

Answer any 6 questions. Each question carries 5 marks.

Q1.Create an HTML page with a temperature converter (Celsius to Fahrenheit and vice versa). Use input field, and convert button. Implement using function expressions and demonstrate. Style inputs and results nicely.

[5 Marks]

Q2. Build a color palette generator. Have a button that generates 5 random colors (using Math functions). Display them as colored divs with hex codes. Allow clicking a color to copy to clipboard. Use an object to track selected colors (Set for uniqueness). (Focus: DOM manipulation, Objects, Set, Events)

[5 Marks]

Q3. Implement a counter component with increment, decrement, and reset buttons. Add a feature to set a custom step value. Persist count using localStorage. Use let/const properly and show understanding of block scope inside event handlers. Animate count change with CSS. (Focus: Variables/Scope, localStorage, Event Handling)

[5 Marks]

Q4. Create a searchable dropdown using a datalist or custom div list. Populate it with an array of programming topics from the course (e.g., DOM, Functions, Arrays). On selection, display detailed explanation fetched from a Map. Style as modern select. (Focus: Arrays, Map, DOM Events, UI components)

[5 Marks]

Q5. Design a simple form for student registration. Use try-catch for validating email format and age (>18). On successful submission, add to an array and display in a table. Clear form after submit. Use CSS for table styling and validation error messages. (Focus: Error Handling, Arrays, Forms, DOM)

[5 Marks]

Q6. Build tabs interface (Home, About, Contact) using HTML divs and JS. Switching tabs should load different content (use objects for tab data). Implement smooth fade transition with CSS. (Focus: Objects, Event delegation, CSS + JS integration)

[5 Marks]

Part B (30 Marks)

Answer all 3 questions. Each carries 10 marks.

Q7. Develop a "Flashcard Learning App" for JavaScript concepts. Have an array of question-answer objects. Show question, flip to answer button (CSS 3D flip effect), next card, and track progress. Implement rating system (easy/hard) using localStorage to prioritize cards. Include error handling for empty deck. (Focus: Objects/Arrays, DOM + CSS Animation, localStorage, State management)

[10 Marks]

Q8. Create an Image Gallery with search and filter functionality. Store image metadata (url, tags, likes) in an array of objects. Allow liking (use Set for liked images). Implement infinite-scroll simulation or "Load More" with array slicing. Use CSS Grid for responsive gallery. Handle search across tags. (Focus: Arrays, Set/Map, Filtering, Advanced UI, Event Handling)

[10 Marks]

Q9. Build a "Shopping Cart Simulator". Display products from an array. Add to cart functionality with quantity. Show cart sidebar with total calculation. Use OOP principles (constructor for Product and Cart). Persist cart. Implement discount code validation (try-catch + conditionals). Style as e-commerce UI. (Focus: OOP, Arrays/Objects, Complex State, Error Handling, UI Integration)

[10 Marks]
